# crawler

Nodejs crawler for [cnbeta.com](http://www.cnbeta.com/)

- 用于爬取并保存cnbeta新闻内容及图片
- 支持爬取条数限制，默认50条
- 支持301跳转检测
- 仅用于Nodejs学习，无意冒犯

## 使用

- `npm install`
- `node app [limitNumber=50]`

## 示例

`node app 10`
